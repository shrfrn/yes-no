Yes-No

https://ca-yesno.herokuapp.com/api/yes-no/ returns an object with the following structure:

        {
            "answer": "yes",
            "image": "https://i.giphy.com/media/GCvktC0KFy9l6/giphy.webp"
        }

or

        {
            "answer": "no",
            "image": "https://i.giphy.com/media/JtLrtaN4VPoKXJRKGB/giphy.webp"
        }

"answer" will randomly be either yes or no

"image" will point to an appropriate animated GIF
